
# Wisherr Frontend

Application React (TypeScript) pour la plateforme Wisherr.

## Démarrage rapide

1. Copier le `.env` à la racine du projet et adapter la variable `REACT_APP_DEFAULT_LOCALE` si besoin. Toutes les variables d'environnement frontend sont centralisées à la racine.
2. Installer les dépendances :

```bash
npm install
```

3. Lancer le serveur de développement :

```bash
npm start
```

## Stack
- React + TypeScript
- Material UI
- Zustand (gestion d’état)
- react-i18next (i18n)
- Axios (requêtes API)

## Fonctionnalités prévues
- Authentification JWT
- Gestion des listes, articles, réservations
- Notifications internes
- Internationalisation (français/anglais)
